# VisionMax

VisionMax is a Flutter-based assistive application designed to help visually impaired users navigate their surroundings safely. The app uses **Google ML Kitâ€™s Object Detection API** to detect nearby objects and provides **audio and haptic feedback** to guide users.

---

## Key Features

- Real-time object detection using device camera  
- Audio alerts announcing detected objects  
- Haptic feedback for safe navigation  
- Lightweight and optimized for fast, offline performance
